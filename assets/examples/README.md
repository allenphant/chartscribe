# 風格預設範例圖片

把每個風格預設的範例輸入圖放在這個資料夾，檔名對應 `js/examples.js` 裡的 `image` 欄位：

| 預設 | 檔名 |
|------|------|
| 通用 | `general.png` |
| AMR－敏感性圖譜 | `amr_susceptibility.png` |
| AMR－抗藥性百分比橫條圖 | `amr_resistance_bar.png` |
| AMR－菌株組成堆疊長條圖 | `amr_composition.png` |
| AMR－多重抗藥趨勢折線圖 | `amr_trend.png` |

建議用該預設最典型的一張圖表截圖。檔案不存在時，卡片會顯示「範例圖」佔位，不影響功能。

對應的「範例產出文字」在 `js/examples.js` 的 `output` 欄位，直接改字串即可。
